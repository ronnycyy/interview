### TD-CLI
为团队量身定制的前端脚手架工具

#### 功能
1. 快速创建模板项目: react+redux+ts
2. 快速生成 action/reducer:  只需一个 action 名称，生成 redux 的所有配置
3. 快速生成 *.model.ts:  基于 quicktype 通过一次 HTTP 响应，自动生成 model 结构

#### 技术
1. Node.js
